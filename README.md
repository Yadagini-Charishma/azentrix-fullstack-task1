## Overview

Budget Tracker is a full-stack web application developed using Spring Boot and MySQL. It helps users manage their expenses efficiently by allowing them to add, update, delete, and track daily expenses. The application provides a simple and user-friendly interface for monitoring personal spending and managing financial records.

---

## Features

* Add new expenses
* View all expenses
* Update existing expenses
* Delete expenses
* Track expenses by category
* Dashboard with expense overview
* Monthly expense summary
* MySQL database integration
* Responsive user interface using Thymeleaf

## How to Run the Project

### Clone Repository

git clone <repository-url>

### Navigate to Project

cd budget-tracker

### Configure MySQL Database

Update application.properties with your MySQL credentials.

### Run Application

Using Spring Tool Suite (STS):

* Right Click Project
* Run As
* Spring Boot App

### Access Application

http://localhost:8080
